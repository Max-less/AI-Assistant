# Основы Scrum

Scrum — это фреймворк для управления проектами, основанный на принципах Agile.

## Роли в Scrum

- **Product Owner** — владелец продукта, определяет приоритеты задач.
- **Scrum Master** — фасилитатор процесса, устраняет препятствия.
- **Development Team** — команда разработки (3–9 человек).

## Артефакты

1. **Product Backlog** — упорядоченный список всех требований к продукту.
2. **Sprint Backlog** — набор задач, выбранных для выполнения в спринте.
3. **Increment** — результат спринта, потенциально готовый к выпуску продукт.

## Церемонии

- Sprint Planning — планирование спринта (до 8 часов для месячного спринта).
- Daily Standup — ежедневная встреча (15 минут).
- Sprint Review — демонстрация результатов спринта.
- Sprint Retrospective — ретроспектива, анализ процесса.
